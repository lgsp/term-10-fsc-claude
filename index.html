Cercle trigonométrique

On enroule la droite réelle sur le cercle de centre \(O\) et de rayon 1. Le point d'enroulement de l'abscisse \(\theta\) est le point \(M(\cos\theta\,;\,\sin\theta)\) du cercle :

\[\cos^2\theta + \sin^2\theta = 1\quad\forall\theta\in\mathbb{R}\] C'est l'identité trigonométrique fondamentale.

03

Propriétés de sin et cos

Périodicité

\(\cos\) et \(\sin\) sont périodiques de période \(2\pi\) :
\[\forall\theta\in\mathbb{R},\quad\cos(\theta+2\pi)=\cos\theta,\quad\sin(\theta+2\pi)=\sin\theta\]

Parité

• \(\cos\) est paire : \(\cos(-\theta)=\cos\theta\)

• \(\sin\) est impaire : \(\sin(-\theta)=-\sin\theta\)

Symétries — angles remarquables

• \(\cos(\pi-\theta)=-\cos\theta\) ; \(\sin(\pi-\theta)=\sin\theta\)

• \(\cos(\pi+\theta)=-\cos\theta\) ; \(\sin(\pi+\theta)=-\sin\theta\)

• \(\cos\!\left(\frac{\pi}{2}-\theta\right)=\sin\theta\) ; \(\sin\!\left(\frac{\pi}{2}-\theta\right)=\cos\theta\) (complémentarité)

• \(\cos\!\left(\frac{\pi}{2}+\theta\right)=-\sin\theta\) ; \(\sin\!\left(\frac{\pi}{2}+\theta\right)=\cos\theta\)

Formules d'addition et de duplication

\[\cos(a+b) = \cos a\cos b - \sin a\sin b\] \[\cos(a-b) = \cos a\cos b + \sin a\sin b\] \[\sin(a+b) = \sin a\cos b + \cos a\sin b\] \[\sin(a-b) = \sin a\cos b - \cos a\sin b\]
Démonstration géométrique de cos(a+b) = cos a·cos b − sin a·sin b
Considérons les points \(M(\cos a;\sin a)\) et \(N(\cos(-b);\sin(-b))=(\cos b;-\sin b)\) sur le cercle unité.

La distance \(MN^2\) vaut (par Pythagore) :
\(MN^2=(\cos a-\cos b)^2+(\sin a+\sin b)^2=2-2(\cos a\cos b-\sin a\sin b)\)

D'autre part, l'angle \(\widehat{MON}=a+b\), et par la loi des cosinus dans le triangle \(MON\) (avec \(OM=ON=1\)) :
\(MN^2=1+1-2\cos(a+b)=2-2\cos(a+b)\)

En égalisant : \(\cos(a+b)=\cos a\cos b-\sin a\sin b\). \(\blacksquare\)
\[\cos(2a) = \cos^2 a - \sin^2 a = 2\cos^2 a - 1 = 1 - 2\sin^2 a\] \[\sin(2a) = 2\sin a\cos a\]
\[\cos^2 a = \frac{1+\cos(2a)}{2}\] \[\sin^2 a = \frac{1-\cos(2a)}{2}\] \[\cos a\cos b = \frac{\cos(a-b)+\cos(a+b)}{2}\] \[\sin a\sin b = \frac{\cos(a-b)-\cos(a+b)}{2}\] \[\sin a\cos b = \frac{\sin(a+b)+\sin(a-b)}{2}\]

05

Dérivées de sin et cos

\[(\sin x)' = \cos x\] \[(\cos x)' = -\sin x\] \[(\sin u)' = u'\cos u\] \[(\cos u)' = -u'\sin u\]
Démonstration — \((\sin x)' = \cos x\)
Par définition de la dérivée, avec la formule d'addition :

\(\dfrac{\sin(x+h)-\sin x}{h}=\dfrac{\sin x\cos h+\cos x\sin h-\sin x}{h}\)

\(=\sin x\cdot\dfrac{\cos h-1}{h}+\cos x\cdot\dfrac{\sin h}{h}\)

Quand \(h\to 0\) : \(\dfrac{\sin h}{h}\to 1\) et \(\dfrac{\cos h-1}{h}\to 0\) (résultats classiques).

Donc \((\sin x)'=\sin x\cdot 0+\cos x\cdot 1=\cos x\). \(\blacksquare\)

Exemples de dérivation

• \(f(x)=\sin(3x)\) : \(u=3x\), \(u'=3\) → \(f'(x)=3\cos(3x)\)

• \(f(x)=\cos(x^2)\) : \(u=x^2\), \(u'=2x\) → \(f'(x)=-2x\sin(x^2)\)

• \(f(x)=\sin^2 x=(\sin x)^2\) : règle de la chaîne → \(f'(x)=2\sin x\cos x=\sin(2x)\)

• \(f(x)=e^{\sin x}\) → \(f'(x)=\cos x\cdot e^{\sin x}\)

• \(f(x)=x\cos x\) : règle du produit → \(f'(x)=\cos x - x\sin x\)

\[\int\cos x\,\mathrm{d}x = \sin x + C\] \[\int\sin x\,\mathrm{d}x = -\cos x + C\]

Résolution d'équations trigonométriques

Pour \(|a|\leq 1\), les solutions de \(\cos\theta=a\) sont : \[\theta = \pm\arccos(a) + 2k\pi,\quad k\in\mathbb{Z}\]
Pour \(|a|\leq 1\), les solutions de \(\sin\theta=a\) sont : \[\theta = \arcsin(a) + 2k\pi\quad\text{ou}\quad\theta = \pi-\arcsin(a)+2k\pi,\quad k\in\mathbb{Z}\]

• \(\cos\theta=1\) → \(\theta=2k\pi\)

• \(\cos\theta=-1\) → \(\theta=\pi+2k\pi=(2k+1)\pi\)

• \(\cos\theta=0\) → \(\theta=\frac{\pi}{2}+k\pi\)

• \(\sin\theta=0\) → \(\theta=k\pi\)

• \(\sin\theta=1\) → \(\theta=\frac{\pi}{2}+2k\pi\)

• \(\sin\theta=-1\) → \(\theta=-\frac{\pi}{2}+2k\pi=\frac{3\pi}{2}+2k\pi\)

Fonctions \(A\cos(\omega t+\varphi)\) — signaux

La fonction \(f(t)=A\cos(\omega t+\varphi)\) représente un signal sinusoïdal où :
  • \(A>0\) : amplitude (max/min de \(f\))
  • \(\omega>0\) : pulsation (rad/s ou rad/unité)
  • \(T=\dfrac{2\pi}{\omega}\) : période
  • \(\varphi\in\mathbb{R}\) : phase à l'origine

• \(f\) est périodique de période \(T=2\pi/\omega\)

• \(-A\leq f(t)\leq A\) (amplitude bornée)

• \(f'(t)=-A\omega\sin(\omega t+\varphi)\) (dérivée = signal de même pulsation, déphasé de \(\pi/2\))

Démonstrations exemplaires du programme

Démo 1 — \(\cos^2\theta+\sin^2\theta=1\)

Par définition, \(M(\cos\theta;\sin\theta)\) est le point du cercle unité de centre \(O\) associé à l'angle \(\theta\). Comme \(M\) est sur ce cercle de rayon 1 : \[OM^2=\cos^2\theta+\sin^2\theta=1^2=1\quad\blacksquare\]

Démo 2 — \((\sin x)'=\cos x\) (voir section 05)

Utilise \(\frac{\sin h}{h}\to 1\) et \(\frac{\cos h-1}{h}\to 0\) quand \(h\to 0\), et la formule d'addition de \(\sin\).

Démo 3 — \((\cos x)'=-\sin x\)

Méthode 1 (formule d'addition) : Même calcul que pour \((\sin x)'\) avec \(\cos(x+h)=\cos x\cos h-\sin x\sin h\) :

\(\dfrac{\cos(x+h)-\cos x}{h}=\cos x\cdot\dfrac{\cos h-1}{h}-\sin x\cdot\dfrac{\sin h}{h}\to\cos x\cdot 0-\sin x\cdot 1=-\sin x\). \(\blacksquare\)

Méthode 2 (parité/translation) : \(\cos x=\sin(\pi/2-x)\), donc \((\cos x)'=-(\sin(\pi/2-x))'=-\cos(\pi/2-x)\cdot(-1)=-\cos(\pi/2-x)=-\sin x\). \(\blacksquare\)

Démo 4 — Formule de duplication \(\sin(2a)=2\sin a\cos a\)

Application directe de la formule d'addition \(\sin(a+b)=\sin a\cos b+\cos a\sin b\) avec \(b=a\) :

\(\sin(a+a)=\sin a\cos a+\cos a\sin a=2\sin a\cos a\). \(\blacksquare\)

Démo 5 — \(\cos(2a)=2\cos^2a-1\)

De \(\cos(2a)=\cos^2a-\sin^2a\) et \(\sin^2a=1-\cos^2a\) :

\(\cos(2a)=\cos^2a-(1-\cos^2a)=2\cos^2a-1\). \(\blacksquare\)
